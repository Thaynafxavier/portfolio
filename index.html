Skills

Minhas Habilidades

  • HTML

    HTML é a linguagem de marcação usada para criar e organizar o conteúdo de páginas da web. Utiliza tags para marcar elementos e permite aos navegadores interpretarem e exibirem corretamente o conteúdo. É a base fundamental de uma página da web, enquanto outras tecnologias são usadas para estilização e interatividade.

  • JavaScript

    JavaScript é uma linguagem de programação usada para adicionar interatividade e dinamismo às páginas da web. Ela permite manipular elementos HTML, responder a eventos, validar formulários, fazer requisições a servidores e muito mais. É essencial no desenvolvimento web, tornando as páginas mais interativas e responsivas, tanto no lado do cliente quanto no servidor.

  • CSS

    CSS é uma linguagem de estilo que controla a aparência e o layout dos elementos em uma página da web. Permite definir cores, tamanhos, fontes e posicionamentos dos elementos. É usado para separar o conteúdo do design visual, facilitando a manutenção e tornando as páginas mais atraentes.

  • CYPRESS

    Cypress é uma ferramenta de teste automatizado para desenvolvimento web. Permite criar testes interativos que simulam as ações do usuário em um navegador, facilitando a garantia de qualidade e o desempenho das aplicações.

  • API

    API (Application Programming Interface) é um conjunto de regras que permite a comunicação entre diferentes softwares. Ela define como os aplicativos podem interagir e compartilhar dados de maneira eficiente. As APIs fornecem uma interface padronizada para que os desenvolvedores integrem e utilizem funcionalidades de outros sistemas de forma segura e confiável.

  • Jira Software e Confluence

    Jira Software é uma plataforma de gerenciamento de projetos e rastreamento de problemas, enquanto o Confluence é uma plataforma de colaboração e documentação de equipes. O Jira Software ajuda a planejar e acompanhar projetos, enquanto o Confluence facilita a criação e compartilhamento de conteúdo colaborativo.
